# Calendar

A **fully-responsive**, **accessible** date-picker component built on
[MUI X `DateCalendar`](https://mui.com/x/react-date-pickers/date-calendar/)
with the `dayjs` adapter.

---

## Overview

The `Calendar` component supports three selection modes via the `mode` prop:

| Mode     | Description                           |
| -------- | ------------------------------------- |
| `single` | Pick one date at a time (default)     |
| `range`  | Pick a start and an end date          |
| `multi`  | Toggle any number of individual dates |

---

## Installation

```bash
# Already in your project ‚Äî just add MUI X and dayjs:
npm install @mui/x-date-pickers dayjs
```

> Your project already ships `@mui/material` v7 and `@emotion/*`. No extra peer deps needed.

---

## Quick Start

```tsx
import dayjs from 'dayjs';
import { Calendar } from '@/components/Calendar';

// Single date
<Calendar
  mode="single"
  value={dayjs()}
  onChange={(date) => console.log(date?.format('YYYY-MM-DD'))}
/>

// Date range
<Calendar
  mode="range"
  value={{ start: dayjs(), end: dayjs().add(7, 'day') }}
  onChange={({ start, end }) => console.log(start, end)}
/>

// Multi date
<Calendar
  mode="multi"
  value={[dayjs(), dayjs().add(3, 'day')]}
  onChange={(dates) => console.log(dates.map((d) => d.format('YYYY-MM-DD')))}
/>
```

## Prop Reference

### Common props (all modes)

| Prop                | Type                             | Default       | Description                               |
| ------------------- | -------------------------------- | ------------- | ----------------------------------------- |
| `mode`              | `'single' \| 'range' \| 'multi'` | ‚Äî             | **Required.** Selection mode.             |
| `minDate`           | `Dayjs`                          | ‚Äî             | Earliest selectable date.                 |
| `maxDate`           | `Dayjs`                          | ‚Äî             | Latest selectable date.                   |
| `shouldDisableDate` | `(day: Dayjs) => boolean`        | ‚Äî             | Predicate to disable individual days.     |
| `fullWidth`         | `boolean`                        | `true`        | Stretch to 100% of container width.       |
| `aria-label`        | `string`                         | mode-specific | Accessible label for the calendar widget. |

### `mode="single"` specific props

| Prop       | Type                            | Description               |
| ---------- | ------------------------------- | ------------------------- |
| `value`    | `Dayjs \| null`                 | Controlled selected date. |
| `onChange` | `(date: Dayjs \| null) => void` | Fired on day click.       |

### `mode="range"` specific props

| Prop       | Type                                           | Description             |
| ---------- | ---------------------------------------------- | ----------------------- |
| `value`    | `{ start: Dayjs \| null; end: Dayjs \| null }` | Controlled range.       |
| `onChange` | `(range: DateRange) => void`                   | Fired after each click. |

### `mode="multi"` specific props

| Prop       | Type                       | Description                         |
| ---------- | -------------------------- | ----------------------------------- |
| `value`    | `Dayjs[]`                  | Controlled array of selected dates. |
| `onChange` | `(dates: Dayjs[]) => void` | Fired after each toggle.            |

## Accessibility

- Day grid uses `role="grid"` / `role="gridcell"` ‚Äî screen-reader friendly.
- Arrow keys (`‚Üê ‚Üí ‚Üë ‚Üì`) move focus between day cells.
- Navigation arrows are reachable via `Tab` and operable with `Enter` / `Space`.
- All interactive elements display a `:focus-visible` ring styled from the MUI theme.
- Disabled days carry `aria-disabled="true"` and are skipped by keyboard navigation.
- Component passes `axe-core` / WCAG 2.1 AA checks (via `@storybook/addon-a11y`).

## Responsive Behaviour

| Viewport | Layout                                                   |
| -------- | -------------------------------------------------------- |
| 320 px   | Full-width, day cells ‚â• 44 √ó 44 px (`2.75rem`)           |
| 375 px   | Full-width, comfortable spacing                          |
| 768 px   | Constrained to `theme.spacing(48)`, centred in container |
| 1024 px+ | Inline within parent layout, same max-width constraint   |

All sizing is expressed in `theme.spacing()` or `rem` ‚Äî no hardcoded `px` values.
