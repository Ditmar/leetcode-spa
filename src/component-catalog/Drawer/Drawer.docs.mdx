# Drawer

Responsive Drawer component built using Material UI Drawer and SwipeableDrawer primitives.

## Features

- Left, right, top and bottom anchors
- Temporary, persistent and permanent variants
- Mobile swipe support
- Responsive bottom sheet
- Optional close button
- Accessibility support
- Theme token support
- Figma-aligned desktop and mobile layouts
