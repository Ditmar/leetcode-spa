# Form Component

The `Form` component is a dynamic and reusable solution for building forms using **React Hook Form** and **Material UI**. It allows rendering different types of fields based on configuration.

---

## ✨ Features

- Supports multiple field types:
  - `input` (text, number, search, etc.)
  - `select`
  - `radio`
  - `checkbox`
  - `textarea`
- Integrated with `react-hook-form`
- Schema-based validation
- Automatic default values handling
- Supports `disabled` state
- Responsive layout with grid system
