# Avatar

The `Avatar` component is used to represent users by their initials. 
It supports multiple sizes and color variants based on the design system.

### Features
- Support for 4 sizes: **sm, md, lg, xl**. [cite: 36, 37, 38, 39]
- Theme-based variants: **default, primary, success**. [cite: 21]
- Responsive circular shape. [cite: 34]
