# Collapsible

Componente reactivo basado en **Radix UI** y **MUI System**. Permite organizar contenido en listas colapsables con un diseño de tarjetas limpio.

### Guía de Uso
1. Se integra con el `theme` de MUI para espaciados consistentes.
2. Utiliza `gap` para la separación de los items internos.
