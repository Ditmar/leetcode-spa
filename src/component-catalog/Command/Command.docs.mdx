# Command

A command palette component built with MUI for searching and executing grouped actions.

## Features

- Grouped command items
- Real-time search
- Optional icons
- Optional keyboard shortcuts
- Keyboard navigation
- Empty state
- Responsive dialog

## Behavior

- `ArrowDown` and `ArrowUp` move through items
- `Enter` selects the active item
- `Escape` closes the palette
- Typing filters results in real time

## Accessibility

Supports combobox behavior and keyboard navigation with ARIA-friendly attributes.
