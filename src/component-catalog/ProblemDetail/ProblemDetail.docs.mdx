# ProblemDetail

The `ProblemDetail` component is a responsive page-level content panel inspired by coding challenge platforms such as LeetCode. It is responsible for presenting complete problem information including title, difficulty, tags, examples, explanations, and constraints while maintaining accessibility, responsiveness, and consistency with the project's Material UI design system.

---

# Purpose

This component was designed to provide a clean and readable interface for algorithmic problem descriptions. The layout prioritizes readability, semantic structure, and responsive behaviour while integrating fully with the project's design token architecture.

The component supports:
- Coding challenge descriptions
- Example rendering
- Constraint visualization
- Difficulty indicators
- Responsive layouts
- Accessible semantic structure
