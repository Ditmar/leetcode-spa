# Menubar

The **Menubar** component provides a responsive navigation system built with Material UI. It displays a horizontal menu on desktop and collapses into a hamburger menu with a drawer on mobile devices.

---

## Features

- Horizontal navigation bar for desktop
- Dropdown menus using Material UI `Menu`
- Mobile responsive design with `Drawer`
- Keyboard navigation support
- Accessible (ARIA roles supported)
- Supports icons and nested menu items

---

## Usage

```tsx
import { Menubar } from '@/component-catalog/Menubar';

const items = [
  {
    label: 'File',
    children: [
      { label: 'New' },
      { label: 'Open' },
    ],
  },
  {
    label: 'Edit',
    children: [
      { label: 'Undo' },
      { label: 'Redo' },
    ],
  },
];

<Menubar items={items} />;
