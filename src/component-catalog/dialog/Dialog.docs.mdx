# Dialog Component

The `Dialog` component is a modal overlay that requires user attention. It adapts to all screen sizes, becoming full-screen on mobile devices.

## Features

- **Configurable Size:** Supports sm, md, lg, xl.
- **Persistent Mode:** Prevents closing on backdrop click or Escape key when enabled.
- **Responsive:** Automatically switches to full-screen mode on screens smaller than 600px.
