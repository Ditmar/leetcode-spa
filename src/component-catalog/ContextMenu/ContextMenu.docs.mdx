# ContextMenu

A context-sensitive menu component built on MUI's `Menu` family. Opens on
**right-click** (desktop) or **long-press ≥ 600ms** (mobile/touch) and always
positions itself within the viewport.

---

## Overview

The `ContextMenu` component wraps any element and attaches context menu behavior
to it. It supports grouped items with dividers, icons, disabled states, and full
keyboard navigation.

---

## Quick Start

```tsx
import { ContextMenu } from '@/component-catalog/ContextMenu';

const groups = [
  {
    id: 'clipboard',
    items: [
      { id: 'copy', label: 'Copy', onClick: () => {} },
      { id: 'paste', label: 'Paste', onClick: () => {} },
    ],
  },
];

function Example() {
  return (
    <ContextMenu groups={groups}>
      <div>Right click here</div>
    </ContextMenu>
  );
}
```

## Behaviour

### Desktop
Opens on the native `contextmenu` event (right-click or keyboard menu key).
The browser's default context menu is suppressed via `event.preventDefault()`.

### Mobile / Touch
A `pointerdown` event starts a 600ms timer. If the pointer is held without
being released or cancelled, the menu opens at the pointer position. Releasing
early cancels the timer.

### Viewport Clamping
The position is clamped with `clampToViewport()` before rendering so the menu
never overflows the right or bottom edges of the screen at any breakpoint.

### Closing
The menu closes on item selection, click outside, or `Escape` key.

## Accessibility

- Menu root renders with `role="menu"`.
- Each item renders with `role="menuitem"`.
- Disabled items expose `aria-disabled="true"` and are skipped by interaction.
- Keyboard navigation (`Escape`) is handled natively by MUI's `Menu`.
- Touch targets meet WCAG 2.5.5 — minimum `44px` height via `theme.spacing(5.5)`.
- Passes `axe-core` / WCAG 2.1 AA checks via `@storybook/addon-a11y`.

---

## Responsive Behaviour

| Viewport | Behaviour                                              |
| -------- | ------------------------------------------------------ |
| 320 px   | Menu clamped to viewport, touch targets ≥ 44px         |
| 375 px   | Full functionality, comfortable spacing                |
| 768 px   | Desktop trigger (right-click) available                |
| 1024 px+ | Full desktop experience, viewport clamping active      |

All sizing uses `theme.spacing()` — no hardcoded `px` values.
