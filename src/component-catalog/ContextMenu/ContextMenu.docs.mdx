# ContextMenu

A context-sensitive menu triggered by right-click (desktop) or long-press ≥ 500ms (mobile).

## Usage

Wrap any element with `<ContextMenu>` and pass a list of groups:
```tsx
<ContextMenu groups={menuGroups}>
  <div>Right-click here</div>
</ContextMenu>
```
