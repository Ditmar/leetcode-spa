# NavigationMenu Documentation

## NavigationMenu.tsx

Primary component file that exports the NavigationMenu component. Renders a responsive navigation menu with desktop and mobile layouts. Uses Material UI AppBar for desktop, Toolbar for content layout, and Drawer for mobile menu. Integrates hooks for state management and utilities for configuration. Implements ARIA attributes (aria-current, role) for accessibility.

## NavigationMenu.types.ts

TypeScript interface definitions ensuring type safety across the component. Defines NavigationMenuProps for component props, NavSection for grouping navigation items, NavItem for individual menu items, and NavigationMenuSize/NavigationMenuVariant for configuration options.

## NavigationMenu.constants.ts

Centralized constants shared across the component. Includes ANIMATION_DURATIONS for consistency, ANIMATION_EASING for smooth transitions, Z_INDEX_STACK for proper layering, TOUCH_TARGET_MIN for accessibility compliance, and NAV_COLORS (BACKGROUND and BORDER) for design tokens.

## NavigationMenu.hook.ts

Custom React hook (useNavigationMenuState) that separates state logic from UI rendering. Manages mobile drawer state, active item detection based on current path, desktop panel state, keyboard navigation handlers, and device detection using useMediaQuery. Returns all necessary state and handlers for the component.

## NavigationMenu.styles.ts

Styled components created with Material UI's styled() API. Defines StyledAppBar, StyledToolbar, StyledDesktopNav, StyledListItemButton, StyledDrawer, StyledMobileNav, and StyledNavContainer. Uses theme tokens and design system values instead of hardcoded values. Implements responsive breakpoints for mobile/desktop switching.

## NavigationMenu.utils.ts

Utility functions that provide configuration based on props. getSizeConfig returns padding and font sizes for different menu sizes (small, medium, large). getVariantConfig returns background and text colors based on variant (primary, secondary). Keeps component JSX clean and configuration centralized.

## NavigationMenu.stories.tsx

Storybook stories in CSF format for component development and documentation. Includes Default (no logo), WithLogo (responsive logo switching), ActiveState (current path highlighting), SizeSmall/SizeLarge (size variants), VariantSecondary (color variant), and CustomLabels (dynamic text). ResponsiveLogoWrapper demonstrates mobile/desktop responsive patterns.

## NavigationMenu.test.tsx

Unit tests using Vitest and Testing Library. Validates render without crashing, navigation items display, active item aria-current attribute, mobile hamburger rendering, and item click handling. Ensures component behavior consistency across different viewport sizes.
