# Progress Component

The `Progress` component provides visual feedback about the status of an ongoing process or task.

It supports both linear and circular indicators, determinate and indeterminate modes, and integrates with the design system theme tokens.

---

## Features

- Linear and Circular variants
- Determinate and Indeterminate modes
- Optional progress label
- Theme-based colors and sizing
- Responsive layout
- Accessible (ARIA compliant)
- Customizable thickness and size for circular indicators

---

## Usage

```tsx
<Progress
  variant="linear"
  mode="determinate"
  value={70}
  showLabel
/>
```

## Accessibility

The `Progress` component follows accessibility best practices:

- Uses native MUI progress components with ARIA support
- Supports determinate and indeterminate semantics
- Labels improve readability for assistive technologies
- Color usage should maintain sufficient contrast ratios

---

## Responsive Behavior

- The component adapts to the width of its container
- Circular indicators scale using theme spacing tokens
- Suitable for dashboards, cards, forms, and loading states
