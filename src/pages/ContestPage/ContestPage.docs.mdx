# ContestPage — Page-Level Contest Dashboard

The `ContestPage` is a page-level organism component built with Material UI (MUI), TypeScript, and responsive design principles following Component-Driven Development (CDD) architecture. It serves as the main dashboard for developers to explore, register for, and review programming contests.

---

## Basic Usage

```tsx
import ContestPage from './ContestPage';

export default function App() {
  return <ContestPage/>;
}
