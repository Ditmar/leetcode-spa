# NavigationSidebar

The `NavigationSidebar` component renders the vertical navigation used across the dashboard layout. It includes the vertical `Logo`, a list of navigation items (icon + label) and supports highlighting the active route with a gradient for both icon and label.

## Responsibilities

- Render a vertical sidebar with configurable `items`.
- Expose `onNavigate` so parent can handle routing.
- Support `currentPath` to control which item is active.

## Accessibility

- Items are keyboard-focusable via `ListItemButton`.
- Use proper `aria-label` on Logo or top container when needed.

## Usage

```tsx
import NavigationSidebar from 'src/components/NavigationSidebar/NavigationSidebar';

<NavigationSidebar
	onNavigate={(r) =&gt; navigate(r)}
	currentPath="/dashboard"
/>
```
