# LeaderboardPage

Componente de página para mostrar un ranking de usuarios. Implementa:

- Vista de tabla (desktop) y cartas (mobile).
- Resalta al usuario actual.

## Props

- `currentUserId?: string` — id del usuario actual para destacar su fila.

## Data shape

```ts
type LeaderboardUser = {
  id: string;
  rank: number;
  avatarUrl?: string;
  fullName: string;
  username: string;
  points: number;
  testsPassed: number;
  lastActiveAt?: string; // ISO date
  category?: string;
};
```

## Behavior

- La página carga datos desde una API simulada (`mockData.fetchLeaderboard`).
- Mientras carga muestra un spinner; en error muestra mensaje.
- Recomendado: desacoplar fetch en un servicio real para producción.
