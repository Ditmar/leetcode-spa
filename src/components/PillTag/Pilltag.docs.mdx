# PillTag

Reusable pill/tag component to display categories like "Courses" or "Test" in a rounded chip style.

---

## Purpose

The **PillTag** component is designed to represent categories like “Courses” or “Test” with a distinct visual style based on Figma designs.  
It follows principles of:

- **Component-Driven Development (CDD)**
- **Pixel-perfect design** according to Figma
- **WCAG AA accessibility**
- **Extensibility and consistency with MUI**

## Key Features

- **Two visual variants:** `primary` and `secondary` (defined in constants)
- **Pixel-perfect design**: exact dimensions, border radii, and shadows
- **Linear gradient text:** from `#B23DEB` to `#DE8FFF` using WebKit
- **White background with shadow:** clean and modern elevation effect
- **Asymmetric borders:** reflecting each variant’s personality
- **Automatic truncation:** long labels shorten with `...`
- **MUI compatibility:** extends `ChipProps` for full flexibility
- **Integrated accessibility:** keyboard support, `focus-visible`, ARIA
- **Robust testing:** 14 Vitest + Testing Library tests
- **Safe fallbacks:** empty labels or invalid variants handled gracefully
