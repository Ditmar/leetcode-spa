The **TestCard** component displays an image (or React node) with a title and description, either overlaid or arranged horizontally.  
It is designed for use cases where you want to highlight visual elements (such as products, users, or courses) with an elegant, accessible, and adaptable appearance.

This component uses `CardActionArea` internally to ensure keyboard accessibility and supports `onSelect` as a standard React mouse event handler.

## âš™ï¸ Main Props

| Property      | Type                               | Description |
|---------------|-----------------------------------|-------------|
| `logo`        | `string \| ReactNode`             | Main image or React node displayed in the card. |
| `title`       | `string`                           | Title displayed in overlay. |
| `description` | `string`                           | Descriptive text shown in overlay. |
| `orientation` | `'vertical' \| 'horizontal'`      | Layout orientation of the card. Defaults to `'vertical'`. |
| `onSelect`    | `React.MouseEventHandler<HTMLButtonElement>` | Function called when the card is clicked, receives the click event. |
| `cardProps`   | `Omit<CardProps, 'onClick'>`       | Optional props to pass to the underlying MUI `Card` component. |
