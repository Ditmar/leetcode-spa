# FilterPanel

El componente `FilterPanel` permite seleccionar un filtro entre varias opciones, mostrando el estado seleccionado y disparando un callback al cambiar.

## Ejemplo b치sico

```tsx
<FilterPanel
  filters={DEFAULT_FILTERS}
  selectedValue="all"
  onSelect={(value) => alert(`Seleccionado: ${value}`)}
/>
```

## Props

| Prop           | Tipo                                      | Descripci칩n                                 |
|----------------|-------------------------------------------|---------------------------------------------|
| filters        | `{ label: string; value: string }[]`       | Lista de opciones de filtro                 |
| selectedValue  | `string`                                  | Valor actualmente seleccionado              |
| onSelect       | `(value: string) => void`                  | Callback al seleccionar un filtro           |
| panelVariant   | `'primary' | 'secondary'`                  | Variante de color del panel (opcional)      |
| size           | `'medium' | 'large' | 'xlarge'`            | Tama침o de los botones (opcional)            |

## Ejemplo con variante secundaria

```tsx
<FilterPanel
  filters={DEFAULT_FILTERS}
  selectedValue="submitted"
  onSelect={(value) => console.log(value)}
  panelVariant="secondary"
/>
```
