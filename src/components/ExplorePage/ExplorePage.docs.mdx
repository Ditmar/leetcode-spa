# ExplorePage

`ExplorePage` is a page-level component for exploring coding interview topics.

## Features

- Page header with title and subtitle
- Stats bar with topics available, problems solved, and overall progress
- Category filter using Material UI `ToggleButtonGroup`
- Difficulty filter using Material UI `ToggleButtonGroup`
- Responsive topic grid
- Topic cards with icon, difficulty chip, title, description, progress bar, category chip, and CTA button
- Empty state when no topics match the selected filters

## Material UI components used

- `Grid`
- `Paper`
- `Typography`
- `Chip`
- `Button`
- `LinearProgress`
- `ToggleButtonGroup`
- `ToggleButton`
- `Skeleton`

## Responsive behavior

- Mobile: 1 column
- Tablet: 2 columns
- Desktop: 3 columns

## Accessibility

Filter buttons are rendered as labelled buttons, and progress bars use determinate values.
